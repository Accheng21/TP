A short description of the project's name and what it does: Human vs AI : Race against an AI through a randomly generated maze and try not to be caught by the AI. You lose if the AI catches you or that it reaches the exit first before you.

How to run the project:: Press control B. 

Which libraries you're using that need to be installed, if any: None

A list of any shortcut commands that exist: Press ‘f’ to have the ai start chasing you, as written on the side of the maze when program is ran. Press ‘r’ to reset everything and generate a new maze
